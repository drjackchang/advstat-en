Topic 12：Repeated Measures ANOVA and GEE

Longitudinal Data Analysis, Sphericity, GEE Correlation Structures, APA Reporting

Learning Objectives

  1. Understand the statistical logic of longitudinal data and repeated measures designs
  2. Correctly execute repeated measures ANOVA
  3. Understand the advantages and appropriate use of GEE
  4. Produce APA-format result tables and reporting sentences

Statistical Method Map (This Topic)

X (Predictor) Y (Outcome) Data Structure Method
Repeated time points Continuous Wide format Repeated Measures ANOVA
Repeated time points Continuous Long format GEE
Repeated time points + Group Continuous Long format Mixed ANOVA / GEE
Note

The core challenge of repeated measures data is non-independence — observations from the same person at different time points are correlated. Standard ANOVA assumes independence; applying it naively to repeated measures data underestimates standard errors and inflates Type I error.

Both repeated measures ANOVA and GEE address this problem, but in different ways.

I. Statistical Theory

1.1 Characteristics of Longitudinal Data

Longitudinal data track the same participants across multiple time points:

ID    Time 1    Time 2    Time 3
001    62.1      68.3      71.5
002    58.4      64.2      69.8
003    70.2      72.1      75.3

Key differences from cross-sectional data:

Cross-Sectional Longitudinal
Data structure One row per person Multiple rows per person
Observations Independent Dependent (same person)
Research question Between-group differences Time trends + between-group differences
Statistical methods t-test, ANOVA Repeated measures ANOVA, GEE, LME

1.2 Repeated Measures ANOVA

Extends one-way ANOVA to repeated measures by partitioning total variance:

\[SS_{\text{Total}} = SS_{\text{Between-subjects}} + SS_{\text{Within-subjects}}\]

\[SS_{\text{Within-subjects}} = SS_{\text{Time}} + SS_{\text{Error}}\]

Key advantage: By separating between-subject variability (SS_Between-subjects) from the error term, the F-statistic denominator becomes smaller, increasing statistical power compared to a standard ANOVA.

Sphericity Assumption: Repeated measures ANOVA assumes that the variances of the differences between all pairs of time points are equal.

Mauchly’s Sphericity Test: - p > .05: Sphericity holds — use standard F-value - p < .05: Sphericity violated → apply Greenhouse-Geisser or Huynh-Feldt correction

1.3 GEE (Generalized Estimating Equations)

Advantages of GEE over repeated measures ANOVA:

Repeated Measures ANOVA GEE
Missing data Complete cases only Can handle missing at random
Outcome type Continuous only Continuous, binary, count
Correlation structure Fixed Specifiable (exchangeable, AR1, etc.)
Estimand Individual-level effect Population-averaged (marginal) effect
Sample size requirement Usable with smaller n Recommend n > 30

Correlation structure options:

  • Exchangeable: All pairs of time points have equal correlation — suitable for repeated measures without strong time ordering
  • AR(1) (autoregressive): Adjacent time points more highly correlated, correlation decays with distance — suitable for ordered longitudinal data
  • Unstructured: No assumption about correlation pattern — most parameters, requires larger sample

1.4 Effect Size

Repeated measures ANOVA: Use partial η²

\[\eta^2_p = \frac{SS_{\text{Time}}}{SS_{\text{Time}} + SS_{\text{Error}}}\]

GEE: Report standardized coefficients or use QIC (quasi-AIC) for model comparison

II. Analysis Workflow

Repeated Measures ANOVA:
Step 1: Confirm wide-format data (one row per participant)
    ↓
Step 2: Descriptive statistics (Mean ± SD per time point)
    ↓
Step 3: Mauchly's sphericity test
    ↓
Step 4: Run repeated measures ANOVA (apply correction if needed)
    ↓
Step 5: Post-hoc pairwise comparisons

GEE:
Step 1: Confirm long-format data (one row per observation)
    ↓
Step 2: Specify correlation structure
    ↓
Step 3: Run GEE
    ↓
Step 4: Interpret β coefficients (population-averaged effects)

                             function(x) x$statistic), 3),

IV. Reporting Results (APA Format)

Repeated Measures ANOVA:

A repeated measures ANOVA examined changes in MIST scores across three time points. The main effect of time was significant, F(2, 398) = 42.31, p < .001, η² = .18 (large effect). Mauchly’s test indicated that the sphericity assumption was met (W = 0.98, p = .21); no correction was applied. Bonferroni-corrected post-hoc comparisons showed that scores at 1-month post-training (M = 69.1) were significantly higher than pre-training (M = 62.1, p < .001); 1-month and 3-month scores did not differ significantly (p = .38), suggesting the training effect peaked at one month and stabilized thereafter.

GEE:

GEE (exchangeable correlation structure) was used to examine MIST score changes across three time points, with pre-training as the reference. Both 1-month post-training (B = 7.02, 95% CI [5.31, 8.73], p < .001) and 3-month post-training (B = 4.88, 95% CI [3.12, 6.64], p < .001) scores were significantly higher than pre-training, indicating a sustained positive training effect. However, the magnitude of improvement was slightly attenuated at 3 months.
