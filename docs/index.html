Course Overview

This course uses R as the primary analytical tool, with EMS and health science contexts as the running thread. Topics range from descriptive statistics to structural equation modeling, with equal emphasis on statistical reasoning and hands-on practice. Each topic includes executable R code, example datasets, and an AI-powered homework feedback system. The final exam is a live coding exam using a single dataset.

Weekly assignments 40%
Participation 20%
