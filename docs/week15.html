Practice 2a: Independent t-Test

t_result <- t.test(stress_vas ~ gender,
                   data      = ems,
                   var.equal = FALSE)

m1 <- mean(ems$stress_vas[ems$gender == "男"])
m2 <- mean(ems$stress_vas[ems$gender == "女"])
s1 <- sd(ems$stress_vas[ems$gender == "男"])
s2 <- sd(ems$stress_vas[ems$gender == "女"])
n1 <- sum(ems$gender == "男")
n2 <- sum(ems$gender == "女")
sp <- sqrt(((n1-1)*s1^2 + (n2-1)*s2^2) / (n1+n2-2))
d_val <- abs((m1 - m2) / sp)

data.frame(
  Comparison = "Male vs Female (Stress VAS)",
  t          = round(t_result$statistic, 3),
  df         = round(t_result$parameter, 1),
  p          = ifelse(t_result$p.value < .001,
                      "< .001",
                      round(t_result$p.value, 3)),
  d          = round(d_val, 3),
  CI         = paste0("[",
                 round(t_result$conf.int[1], 2), ", ",
                 round(t_result$conf.int[2], 2), "]"),
  Effect     = dplyr::case_when(
    d_val >= .80 ~ "Large",
    d_val >= .50 ~ "Medium",
    TRUE         ~ "Small"
  )
) |>
  gt() |>
  tab_header(
    title = "Practice 2a. Independent t-Test: Perceived Stress by Sex"
  ) |>
  tab_source_note(

  ) |>
  cols_label(

    p = "p", d = "d", CI = "95% CI", Effect = "Effect Size"
  ) |>
  cols_align(align = "center",
             columns = c(t, df, p, d, CI, Effect)) |>
  cols_align(align = "left", columns = Comparison) |>
  tab_style(

Practice 2b: One-Way ANOVA

aov_r   <- aov(response_time ~ level, data = ems)
aov_s   <- summary(aov_r)
ss_b    <- aov_s[[1]]$`Sum Sq`[1]
ss_w    <- aov_s[[1]]$`Sum Sq`[2]
eta2    <- ss_b / (ss_b + ss_w)

data.frame(
  Source = c("Between groups (Level)",
             "Within groups (Error)",
             "Total"),
  SS     = round(c(ss_b, ss_w, ss_b + ss_w), 2),
  df     = c(aov_s[[1]]$Df[1],
              aov_s[[1]]$Df[2],
              sum(aov_s[[1]]$Df)),
  MS     = c(round(aov_s[[1]]$`Mean Sq`[1], 2),
              round(aov_s[[1]]$`Mean Sq`[2], 2), NA),
  F      = c(round(aov_s[[1]]$`F value`[1], 3),
              NA, NA),

Practice 3: APA Reporting Templates

Based on the results above, practice writing complete APA-format reporting sentences using these templates:

Independent t-test:

Male EMS personnel (M = X.XX, SD = X.XX) [did / did not] differ significantly from female personnel (M = X.XX, SD = X.XX) in perceived stress, t(df) = X.XX, p = X.XX, 95% CI [X.XX, X.XX], d = X.XX (effect size judgment).

One-Way ANOVA:

A one-way ANOVA showed that response time [did / did not] differ significantly across EMT certification levels, F(df₁, df₂) = X.XX, p [< .001 / = X.XX], η² = X.XX (effect size judgment).

Logistic regression:

Perceived stress [significantly / did not significantly] predict occupational injury, OR = X.XX, 95% CI [X.XX, X.XX], p = X.XX, indicating that [interpretation].

V. Final Exam Information

Important

Final Coding Exam Format (Topic 16)

  • Duration: 120 minutes
  • Dataset: ems_final.csv (distributed on exam day)
  • Tools: R + RStudio; course website (read-only reference)
  • Not permitted: Internet search, AI tools, peer discussion

Required format for each question:

  1. Justification for method selection (X-Y type identification, 1–2 sentences)
  2. Assumption check results
  3. Complete executable R code
  4. Key statistical output
  5. APA-format reporting sentence
  6. Practical interpretation (1 sentence linking results to EMS context)
